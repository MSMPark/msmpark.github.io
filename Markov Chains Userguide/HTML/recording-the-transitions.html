The above code generates markov chains up to a certain number of steps and stores all the markov chains in a matrix. The code is broken up into the three following sections:

  1. Choosing the number of state transitions per trial, and choosing the total number of trials.

  1. Finding the initial state of the system.

  1. Recording and generating a matrix comprised of each trial.

Note: A list of all the trials is contained within the "trials" matrix.
